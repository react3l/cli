@react3l/cli
------------

```sh
Usage: react3l [options] [command]

Options:
  -V, --version   output the version number
  -s, --scss      Generate component along with SCSS file (default: false)
  -n, --native    Generate React Native component (default: false)
  -f, --force     Force to create contents, overwrite any existing entries (default: false)
  -h, --help      display help for command

Commands:
  g:c <name>
  help [command]  display help for command
```
